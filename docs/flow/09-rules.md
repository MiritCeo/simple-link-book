# 09-rules

- Dostepnosc terminu liczona wg:
  - godzin salonu i wyjatkow,
  - przerw `BREAK` i buforow `BUFFER`,
  - grafiku pracownika,
  - kolizji z innymi wizytami.
- Dezaktywacja uslugi/pracownika/klienta nie usuwa danych, tylko wyklucza z aktywnego obiegu.
- Tokeny anulowania sa jednorazowe i czasowe.
